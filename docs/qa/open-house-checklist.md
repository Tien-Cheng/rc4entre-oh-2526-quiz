# Open House Run Checklist

## Before visitors arrive

1. Open the deployed app on laptop/tablet and keep charger connected.
2. Press `H` and verify the host panel opens.
3. Set default mode:
   - Normal crowd: `Hybrid` + `Quiz -> Pitch`
   - High queue: `Pitch Only` or `Quiz Only`
4. Clear leaderboard before event start.
5. Run one full dry round end-to-end.

## During event

1. Keep one facilitator in host mode and one engaging visitors.
2. Use `Start Next Player` from host panel between rounds.
3. Adjust pitch bonus quickly based on pitch quality.
4. If queue builds up, temporarily switch to single-module mode.

## If something goes wrong

1. Press `H` and use `Reset Current`.
2. Refresh browser tab if timer/UI seems stuck.
3. If leaderboard looks wrong, `Clear Board` and continue.
4. Keep a manual backup score sheet for prize tie-breakers.

## End of event

1. Capture top scores for records/prizes.
2. Take screenshots of leaderboard if needed.
3. Close tab and pack booth setup.
